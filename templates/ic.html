Iniciação Científica

Para todo desmonte de rocha feitos por explosivos, existe o projeto do Plano de Fogo que distribui os furos carregados de explosivos de uma maneira geométrica conveniente. No mercado atual, há vários softwares disponíveis para realizar o design desses modelos de Plano de Fogo. O problema é a dificuldade para a utilização desses softwares no contexto acadêmico, pois a maior parte deles possui licença paga. O projeto da iniciação científica tem o intuito de tornar a realização desses projetos mais acessíveis para alunos de graduação/pós-graduação, mais especificamente da Engenharia de Minas.

Linguagem

Para adequar o programa ao uso acadêmico, nós usamos a plataforma de programação Mathematica®, desenvolvido pela empresa Wolfram Research. O principal critério de escolha da linguagem foi a sua facilidade de uso e de acesso, uma vez que a Universidade de São Paulo possui licenças ilimitadas disponíveis para os alunos e docentes.

Resultados

3 vistas diferentes da bancada são exportadas como imagem. Desse modo, fica fácil observar os principais elementos que compõe o Plano de Fogo. Para maiores detalhes, o programa também exporta um relatório sobre todos os dados relevantes de acordo com os dados de entrada inseridos.
